# usage

```console
$ git clone --recursive git@github.com:hekt/dotfiles-instant.git
$ cd dotfiles-instant
$ make initialize # download git contribs
$ make install # make symlinks on $HOME and add lines to $HOME/.{z,ba}shrc
```
